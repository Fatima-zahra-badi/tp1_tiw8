# tp1_tiw8

Pour exécuter le projet :  

1- git clone https://github.com/Fatima-zahra-badi/tp1_tiw8.git  
2- cd tp1_tiw8  
3- npm install(pour installer toutes les dépendances)  
4- npm run build  
5- npm run start( App listening on port: 3000 )
